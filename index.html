Hello world, SASS!!

Installing on Windows

First, you will have to install RubyInstaller.

Run "Start Command Prompt with Ruby" from Start Menu. Then run the below script.

gem install sass

Done, now SASS is on your machine! So, let's play with it. Open your text editor. Create a file "test.scss" along with following contents:

.red{
    color:red;
}

In your Ruby CMD,

$>sass test.scss
.red { 
    color: red; }

It will output the reformatted CSS we wrote above. Let's move on.
